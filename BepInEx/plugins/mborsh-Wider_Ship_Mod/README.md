# WiderShipMod

### [Matty Fixes Experimental](https://thunderstore.io/c/lethal-company/p/mattymatty/Matty_Fixes_Experimental/) is not required for this mod, but you'll probably want to leave it on since it fixes items falling through the floor when loading a save.

### Recommended with:
 + [ShipWindows](https://thunderstore.io/c/lethal-company/p/TestAccount666/ShipWindows/)
 + [2 Story Ship](https://thunderstore.io/c/lethal-company/p/MelanieMelicious/MelanieMelicious_2_sToRy_ShIp__works_w_Wider_Ship_Mod/) - Most of the config of Wider Ship **DOES NOT WORK** with it
 + [Moved Magnet Switch](https://thunderstore.io/c/lethal-company/p/AtomicStudio/Moved_Magnet_Switch/)
 + [Matty Fixes Experimental](https://thunderstore.io/c/lethal-company/p/mattymatty/Matty_Fixes_Experimental/) OR [Matty Fixes](https://thunderstore.io/c/lethal-company/p/mattymatty/Matty_Fixes/)
