# Concentus

Note: This is not my library. This is an upload containing the binaries to allow mods to refer to this as a dependency.  
This package contains the following binaries for netstandard2.0:
- [Concentus v2.2.2](https://www.nuget.org/packages/Concentus)
- [Concentus.Oggfile v1.0.5 - qwbarch fork](https://github.com/qwbarch/concentus.oggfile)

Due to needing long file support when using the briged thunderstore packages, these are also included:
- [System.Memory v4.5.5](https://nuget.org/packages/System.Memory/4.5.5)
- [System.Buffers v4.5.1](https://nuget.org/packages/System.Buffers/4.5.1)
- [System.Numeric.Vectors v4.5.0](https://nuget.org/packages/System.Numerics.Vectors/4.5.0)
- [System.Runtime.CompilerServices.Unsafe v6.0.0](https://nuget.org/packages/System.Runtime.CompilerServices.Unsafe/6.0.0)
