# Blue Merc

This mod adds a model of Blue Merc as a suit that's unlocked at the start of the game. This is a silly mod and my first proper attempt at using the ModelReplacementAPI mod.

* Mod Developed by [Emul](https://www.youtube.com/@ItsEmul)!
* Blue Merc Model made by [Ellis02](https://www.youtube.com/@Ellis02Media).
