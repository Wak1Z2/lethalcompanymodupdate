# MelanieMelicious Utility Furniture Pack v2.0.4
### <i>Wow, I made it a new mod for this title. ONLY UPDATE/ADD ON NEW SAVES! Adds furniture items, many with intended functions such as storage or usable doors. Enabled furniture, shop type, and prices can be changed in configs. You can find me in the Lethal Company Modding Discord.</i>

## Furniture Items:
| Name| Default Price| Luck| Description|
| :-: | :-: | :-: | :-: |
| Workshop | 100 | 0.003 | A more spatial and open storage unit, including a placement wall, for more convenient tool organization and sorting. |
| WorkTable | 60 | 0.003 | Only the bottom section of Workshop, and without the placement wall. |
| WideCabinet | 50 | 0.003 | A wider, doorless version of the vanilla storage closet. More space for storing, and ease-of-access. |
| Wall# [0-14] | 35 | 0 | Wall slots that can be configured to include doors of different styles, to change material type, and shared material color. Does not block enemies. |
| FireExit | 2000 | 0 | Functions as a normal fire exit, but on the ship! Only works with LethalLevelLoader present, and when purchased before next landing. Fire exit mode can be toggled in config (normal, only entrance, or only exit). |
| Bed# [0-3] | 150 | 0.005 | Mostly decorative beds with storage drawers at the sides. Type/style can be configured. |
| Tree# [0-1] | 250 | 0.01 | A tall deciduous tree - mainly intended as outside decor - that has a chance to spawn lower-quality beehives when landed on timed planets! Spawning changes can be toggled in config. |
| Table# [0] | 70 | 0.005 | Decorative bench table. |
| Fern# [0-1] | 40 | 0.008 | Decorative potted ferns. |
| XMASTree0 | 250 | 0.02 | A tall Christmas tree - mainly intended in Wider + 2-Story ship, or as outside decor - with higher chances to spawn presents based on how many items are used to decorate it. Chances per-item can be toggled in config. |
| Fireplace | 100 | 0.01 | Mostly decorative fireplace that emits light. |
