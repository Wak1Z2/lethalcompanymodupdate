# Cudi's Realistic Suits
## About
Adds (currently) 6 new suits that fit the aesthetic of Lethal Company and aid you in organizing and managing large teams.

## Suits
Cudi's Realistic Suits currently adds the following suits:
- Construction
- Maintenance
- Custodian
- Relay
- Squad I (Orange)
- Squad II (Blue)

## Instructions
Manually:
Place the main folder in your BepInEx/plugins folder
Automatic:
Download using mod manager on the website or in the app (ensure that you have thunderstore installed)

Other players must have the mod installed to see/access your suits.
CRS Suits replace the default MoreSuits mod suits. If you wish to change this, remove the "!less_suits" file in "CudisRealisticSuits\plugins\moresuits"
