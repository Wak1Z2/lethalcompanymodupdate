# LoadAllLobby 加载所有大厅

The lobby list will display lobbies from different versions.

加载大厅列表时，将会显示不同版本大厅

# Features 功能

* **Join lobbies across different versions(Only MoreCompany)**
    
    允许跨版本加入大厅(仅MoreCompany)
    * When you install MoreCompany, it allows you to join lobbies where MoreCompany is not installed.
        
        当你安装MoreCompany时，允许你加入没有安装MoreCompany的大厅

* **Load Custom Version Lobbies**  

    加载自定义版本大厅

    * Add the versions you want to load by modifying the `LoadVersion.Versions` configuration.  

        通过修改`LoadVersion.Versions`配置添加你想要加载的版本

* **Adjusted Lobby List Width**  

  调整大厅列表的宽度
  * Lobby names are no longer length-constrained. 

    现在大厅名称不再限制长度  

* **Automatic Lobby Filtering with `LobbyCompatibility`**  

    LobbyCompatibility兼容性

  * When the `LobbyCompatibility` mod is installed, lobbies *without* it will be automatically hidden (and vice versa).  

    当你安装 `LobbyCompatibility` 模组时，会自动隐藏未安装`LobbyCompatibility`模组的大厅（反之亦然）

* **Custom Filter(Support Regex)**

  自定义过滤(支持正则表达式)

    * Hide unwanted lobbies by configuring `LobbyFilter.CustomHide`

      通过修改`LobbyFilter.CustomHide`配置来隐藏你不想看见的房间
